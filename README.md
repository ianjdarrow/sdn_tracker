# sdn_tracker

A simple UI to check client names against the OFAC SDN list

## Build Setup

``` bash
# install dependencies
npm install
pip3 install -r requirements.txt

## live dangerously

./startup.sh

## or, live safely -
# start Sanic API server
python3 app.py

# served with hot reload at localhost:8080
npm run dev
```
